Provides a mock package for Click, enabling unit testing of Pages and Controls.

This package defines the main mock classes but developers only have to be concerned with two classes namely {@link org.apache.click.MockContainer} and {@link org.apache.click.MockContext}.

The main class developers will use is {@link org.apache.click.MockContainer} which provides mocking mocking of application pages and controls.

The class {@link org.apache.click.MockContext} provides mocking of {@link org.apache.click.Control}s and is easier to setup and use than {@link org.apache.click.MockContainer}.
