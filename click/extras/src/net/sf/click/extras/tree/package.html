Provides tree controls for hierarchical data. The main classes are the {@link net.sf.click.extras.tree.Tree} class which provides behavior and rendering and the {@link net.sf.click.extras.tree.TreeNode} class which is the basic building block of a tree's model.

This package also provides a interface called {@link net.sf.click.extras.tree.TreeListener}, that enables users to listen and response to certain tree events.
