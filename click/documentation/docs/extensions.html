Apache Click Extensions

There are a number of extension projects that provides extra features and components.

Below is a list of third party contributions:

  1. ClickClick is a community based extensions project providing many new components and seamless Ajax integration.
  2. Click jQuery is a community based project that aims to integrate Click with jQuery
  3. Click Calendar provides a DHTML popup Calendar component for Click. The Calendar component is based upon the JSCalendar library http://www.dynarch.com/projects/calendar/
  4. Click Charts provides common Chart components for Click. The Chart components are based upon the JavaScript library http://www.walterzorn.com.
  5. Chainer is a simple tool for generating JavaBean subclasses where methods can be chained. You can use Chainer to generate Control subclasses to enable method chaining.
If you have any questions or would like to contribute to these projects you can send an email to our mailing list.
