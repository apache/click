Click Extensions

There are a number of extension projects that provides extra features and components.

Below is a list of third party contributions:

  • Click Calendar provides a DHTML popup Calendar component for Click. The Calendar component is based upon the JSCalendar library http://www.dynarch.com/projects/calendar/
  • Click Charts provides common Chart components for Click. The Chart components are based upon the JavaScript library http://www.walterzorn.com.
Any questions regarding these projects should be raised on one of the Click mailing lists.
