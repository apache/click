Examples

Click provides a number of example web applications to demonstrate features and get you up and running as quickly as possible. These web applications include:

Click Examples
Click Examples (click-examples.war) provides example Pages to demonstrate Click features and programming techniques. This application also allows you to view the Page source code, templates and config files.

View the index page to see whats included. Note you will need to deploy the click-examples.war to try these examples.

Context Path:   /click-examples

Click Blank
Click Blank (click-blank.war) provides a blank application template for starting a new application.

Context Path:   /click-blank

Click Documentation
Click Documentation (click-documentation.war) provides the complete documentation set you are now reading.

Context Path:   /click-documentation


Deploying Examples

To deploy the example web applications:
  1. Ensure JAVA_HOME environment variable is set and points to a JDK installation (1.4 or later).

  2. Ensure ANT_HOME environment variable is set and points to an Ant installation (1.6 or later).

  3. Edit build.properties file and set the dir.appserver property to point to your J2EE web container deployment directory. Also set to dir.tomcat property if you are using Tomcat. For example: 
      # App Server deployment directory
  dir.appserver=C:/java/jakarta-tomcat-5.5.9/webapps 
  
  # Tomcat home directory, for task: setup-tomcat-logging
  dir.tomcat=C:/java/jakarta-tomcat-5.5.9
    Note on Windows ensure you replace any \ directory path characters with / otherwise Ant may not find the directory.

  4. Run the optional Tomcat Ant target: 
      ant setup-tomcat-logging
    This will copy the log4j and commons-logging jars into the tomcat/common/lib directory, and will copy a log4j.properties into tomcat/common/classes. This will provide clearer logging messages than the standard tomcat install.
  5. Run the Ant target: 
      ant deploy-webapps
    This will copy the example application WAR files to your Application Servers deployment directory.

    The example application WAR files come pre-built with the Click distribution. You can also build the examples yourself. See the Building topic for more details.
