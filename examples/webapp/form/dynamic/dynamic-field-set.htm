This example demonstrates how to dynamically add fieldsets to a form.
  • The JavaScript function Click.submit(form, validate) is set as the checkbox onclick handler. Clicking the checkbox will submit the form, but bypass form validation.
  • The utility method ClickUtils.bindAndValidate(Form) is used to bind and inspect the checkbox value before the page onProcess event.
  • If the checkbox is checked, the address fieldset is added to the form
