This AJAX example uses the AutoCompleteTextField control.

${form}

 

The AutoCompleteTextField control uses the 'Script.aculo.us' Ajax.Autocompleter class to make an AJAX request when the users a character value. The page's AutoCompleteTextField control handles this AJAX request and returns the list of suggested String values to render. 
AutoCompleteTextField postCodeField = new AutoCompleteTextField("postCode") {
    public List getAutoCompleteList(String criteria) {
        return getPostCodeService().getPostCodeLocations(criteria);
    }
};
postCodeField.setSize(40);
