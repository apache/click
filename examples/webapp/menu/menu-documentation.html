Application menus is defined in the configuration file /WEB-INF/menu.xml.

The Menu is rendered using the Velocity Macro macro.vm, and is initialized using the JavaScript function initMenu().

The example menu pages all extend MenuPage which defines a "rootMenu" control. This ensures all the example menu pages include the Menu control. 

public class MenuPage extends BorderedPage {
    
    public MenuPage() {
        addControl(new Menu("rootMenu"));
        
        addModel("head-include", "menu/menu.js");
        addModel("body-onload", "initMenu();");
    }
    
}
