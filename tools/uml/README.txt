Apache Click UML Diagrams Project
=================================

This folder contains Apache Click UML diagrams which is drawn with
BOUML (http://bouml.free.fr/).

Usage
=====

Download BOUML (http://bouml.free.fr/) and checkout all files
in this folder to your local machine.

Then simply open the file 'uml-diagrams.prj' using BOUML and you
are ready to go.
