Mail Lists

The following mailing lists are available:
  • user
  • development

User

Here you can ask questions on how to use Click, have discussions around best practices and inquire about new features you would like to see.

  • To subscribe send an empty email to click-user-subscribe@incubator.apache.org
  • To unsubscribe send an empty email to click-user-unsubscribe@incubator.apache.org
  • To post a message use the address: click-user@incubator.apache.org
  • Message archive: archive

Development

This list focuses on development issues such as the future direction, prioritization of issues, design issues and release schedules.

  • To subscribe send an empty email to click-dev-subscribe@incubator.apache.org
  • To unsubscribe send an empty email to click-dev-unsubscribe@incubator.apache.org
  • To post a message use the address: click-dev@incubator.apache.org
  • Message archive: archive
