Click Javadoc

Click is distributed as three separate libraries click.jar, click-extras.jar and click-mock.jar.

The Javadoc API of these libraries is provided below:

  • Click API   -   (click.jar)

    Provides the Core API and consists of everything you need to build a Click web application.

  • Controls API   -   (click.jar)

    Provides the Core Controls, and provides a short description of each control and corresponding HTML element.

  • Extras API   -   (click-extras.jar)

    Provides components that are tailored to specific technologies, such as a Hibernate, Spring and Cayenne. Extras also include many Controls including IntegerField, DoubleField and DateField. You will also find Filters which can improve your application performance.

  • Mock API   -   (click-mock.jar)

    Provides a library for testing Click Pages and Controls, independent of a Servlet container such as Tomcat or Jetty.

You can find these libraries under the /dist folder in the Click distribution you downloaded.
