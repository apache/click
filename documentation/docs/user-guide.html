User Guide

Click comes with a comprehensive user guide to get you started quickly. It starts off with an introductory chapter showing how to work with pages and templates and moves on to some of the most popular controls, Form and Table.

It also explains in depth the Page and Control life cycle, configuration and best practices.

The user guide is available in three flavors:

  • HTML multiple pages
  • HTML single page (320 KB)
  • PDF (480 KB)
