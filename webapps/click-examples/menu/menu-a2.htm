This page provides a demonstration of the Menu component.

 

#writeMenu($rootMenu)

This is page $title.

 

Application menus can be defined in configuration file /WEB-INF/menu.xml

To access the root menu item of the applications configured menus use the static Menu method getRootMenu(): 

public void onInit() {
    Menu rootMenu = Menu.getRootMenu(getContext()); 
    addModel("rootMenu", rootMenu);
}
This method will return copy of the menu tree which can be referened from the root menu item. In the returned tree the Menu item for the current page will be selected, and its parents will also be selected.

In this example the applications menu is rendered as a 2 level menu using a Velocity Macro macro.vm
