Building and Deploying

To build and deploy the web applications:
  1. Ensure JAVA_HOME environment variable is set and points to a JDK installation.

  2. Ensure ANT_HOME environment variable is set and points to an Ant installation.

  3. Edit build.properties file and set the app.server.dir property to point to your J2EE web container deployment directory. Note on Windows ensure you replace any \ directory path characters with / otherwise Ant may not find the directory. For example: 
      # appserver deployment directory
  dir.appserver=C:/java/jakarta-tomcat-5.0.25/webapps/
  4. Run the Ant build script targets: 
      ant build-framework build-webapps deploy-webapps
