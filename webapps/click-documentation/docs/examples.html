Examples

Click provides a number of example web applications to demonstrate features and get you up and running as quickly as possible. These web applications include:

Click Blank
Click Blank (click-blank.war) provides a blank application template for starting a new application.

Local URL: click-blank

Click Documentation
Click Documentation (click-documentation.war) provides the complete documentation set you are now reading.

Local URL: click-documentation

Click Examples
Click Examples (click-examples.war) provides example Pages to demonstate Click features and programming techniques. This application also allows you to view the Page source code, templates and config files.

Local URL: click-examples


Building and Deploying

To build and deploy the web applications:
  1. Ensure JAVA_HOME environment variable is set and points to a JDK installation.

  2. Ensure ANT_HOME environment variable is set and points to an Ant installation.

  3. Edit build.properties file and set the dir.appserver property to point to your J2EE web container deployment directory. For example: 
      # App Server deployment directory
  dir.appserver=C:/java/jakarta-tomcat-5.5.7/webapps
    Note on Windows ensure you replace any \ directory path characters with / otherwise Ant may not find the directory.

  4. Run the Ant build script targets: 
      ant build-framework build-webapps deploy-webapps
    These targets will build the Click framework, then build the example applications, and then copy the WAR files to your Application Servers deployment directory.
