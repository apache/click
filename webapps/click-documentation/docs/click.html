Click

Click is a J2EE web application framework created for commercial Java developers.

Click Highlites

  • Very easy to learn
  • Event base programming model
  • Velocity page rendering
  • High performance

The philosphy behind Click is best summarized in the Swedish word lagom which translates to not too little or too much, but optimal.

The aim of Click is to provide a web application framework which isn't too complicated for commercial development, but also provides enough capability to get the job done.

With Click you should be able get up and running within a day.

Click was developed by Malcolm Edgar and is hosted by SourceForge.net.
