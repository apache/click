Controls

Click provides a number of HTML Controls which support server side processing. These controls are included in the Java package net.sf.click.control.

HTML Controls

The major net.sf.click.html package objects are:
  • ActionLink   -   which provides an <a> href attribute which can call command methods

  • Form   -   which provides Form processing and validation

  • Field   -   which provides Form processing and validation
