Provides a mock package for Click, enabling unit testing of Pages and Controls.

This package defines the main mock classes but developers only have to be concerned with two classes namely {@link net.sf.click.MockContainer} and {@link net.sf.click.MockContext}.

The main class developers will use is {@link net.sf.click.MockContainer} which provides mocking mocking of application pages and controls.

The class {@link net.sf.click.MockContext} provides mocking of {@link net.sf.click.Control}s and is easier to setup and use than {@link net.sf.click.MockContainer}.
