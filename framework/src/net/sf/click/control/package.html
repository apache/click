Provides renderable HTML controls. Controls implement the {@link net.sf.click.Control} interface to perform server side request processing.

Controls are invoked by Forms or by the ClickServlet.

Using these controls is similar to traditional Swing GUI programming. While these controls do not provide a full Swing style MVC implementation. They provided a simplified programming model and high performance, but at the cost of some extensiblity.

HTML controls provide an action callback mechanism similar the java.awt.ActionListener callback. 

public class MyPage extends Page {

    public void onInit() {
        ActionLink actionLink = new ActionLink("link");
        actionLink.setListener(this, "onLinkClick");
        addControl(actionLink);
    }
    
    public boolean onLinkClick() {
        System.out.println("onLinkClick invoked at " + (new Date());
        return true;
    }
}
